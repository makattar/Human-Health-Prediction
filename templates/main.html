Introduction


The main objective of this project is to predict a person’s health from top three deadly diseases. This project works on previous data sets of affected patients and trains the models from itself using one of the Machine Learning algorithm and provide almost accurate prediction.


Major challenge is how to extract the information from these data because the amount is very large so some data mining and machine learning techniques can be used. Also, the expected outcome and scope of this project is that if disease can be predicted than early treatment can begiven to the patients which can reduce the risk of life and save life of patients and cost to get treatment of diseases can be reduced up to some extent by early recognition.


It might feel curious or fascinating that you wanted to know what disease that you might encounter in future or in upcoming times for these personalities and many other organization this system will be very useful topredict one’s health from top three diseases.
